Container

network deque

The following is an interface for a Deque data structure. Generally speaking we call these containers.

Observe how this interface is a templated (i.e. Container(T)), where 'T' is a placeholder for a data type.

interface Container (
T
) {
void push_front(T x);
void push_back(T x);
T pop_front();
T pop_back();
T at(size_t pos);
T back();
T front();
size_t size();
void clear();
}
