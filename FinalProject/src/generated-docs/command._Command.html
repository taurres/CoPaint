Commands that are executed in SDLApp. Command contains methods for executing and unexecuting the command, and convertion between command and packet.

interface Command {
void execute(Surface* surface);
void unexecute(Surface* surface);
Packet toPacket();
static Command fromPacket(Packet packet);
